Create an Array

Array can be created using an array literal or new keyword.
[declaration type] [identifier] = [["String", Number, Boolean, undefined, null, Object, function, Array]]

Access Elements of an Array

We can access array elements using indices Array[[index]].
Array indices start from 0.

Add an Element to an Array

.push() method adds an Element at the end of an Array.
.unshift() method adds an Element at the start of an Array.

Change an Element of an Array

Array[[index]] = value this statement changes an element of an Array.

Remove an Element of an Array

.pop() method removes an Element from the end of an Array.
.shift() method removes an Element from the start of an Array.

These methods also return the element that is removed.

Array Length

[Array].length method return the length of an Array.

Array Methods

Methods that makes changes to array and returns something.

  • [Array].push([element]) adds element to array at end and returns length of the array.
  • [Array].unshift([element]) adds element to array at start and returns length of the array.
  • [Array].pop([element]) removes element of array from end and returns it.
  • [Array].shift([element]) removes element of array from start and returns it.
  • [Array].sort() sorts the array in specific order and return the array.
  • [Array].fill([value], [from index], [to index]) fills array elements with given values and returns the array.

Methods that makes changes to array but returns nothing.

  • [Array].splice([start], [deleteCount], [elements]) removes deleteCount elements from start adds elements at start index.

Methods that doesn't make any changes to array but returns something.

  • [Array].length returns length of an array.
  • [Array].reverse() returns the reversed array.
  • [Array].join() returns a string that is the concatination of the array elements.
  • [Arrsy].toString() returns the string representation of an array.
  • [Array].concat() returns an array that is the concatination of the arrays and elements.
  • [Array].indexOf([element]) returns the first index of occurence of an element.
  • [Array].lastIndexOf([element]) returns the last index of occurence of an element.
  • [Array].slice([from index], [no. of elements]) returns elements from given index to given no. of elements as an array
  • [Array].find([function ()]) returns first element that satisfies a condition that is provided as function.
  • [Array].includes([element]) returns if it includes a value.
  • [Array].reduce([(total, element) => { return [value]; }], [initial value]) reduces array to single value from left using provided function and returns it.
  • [Array].reduceRight([(total, element) => { return [value]; }], [initial value]) reduces array to single value from right using provided function and returns it.
  • Array.isArray([Array]) returns true if array is an array.
  • [Array].filter((element, index, array) => { return [value]; }) returns array by filtering elements on given test.
  • [Array].map((element, index, array) => { return [value]; }) returns array by mapping elements using given function.
  • [Array].forEach((element, index, array) => { return [value]; }) returns array by mapping elements using given function.
  • [Array].some((element, index, array) => { return [condition]; }) returns true if any element passes given condition.
  • [Array].every((element, index, array) => { return [condition]; }) returns true if every element passes given condition.
  • [Array].copyWithin([target], [start], [end]) replaces start to end elements from target element within same array and returns this array
  • [Array].flat([depth]) create a new array with all sub-array elements concatenated into it recursively up to the specified depth.
  • [Array].flatMap((element, index, array) => { return [value]; }, [depth]) equivalent to [Array].map((element, index, array) => { return [value]; }).flat([depth])
  • [Array].toLocaleString([locales], [options])This method converts each array elements to Strings using their toLocaleString methods and separates them by a comma.

Methods that doesn't make any changes to array and returns nothing.

  • [Array].forEach((element, index, array) => { }) loops through on each element of array and executes given function.
