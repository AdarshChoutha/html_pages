Strings

In JavaScript, Strings can be surrounded with 3 types of quotes: Single quotes '', Double quotes "", Backticks ``.
Single quotes and double quotes are practically the same and you can use either of them.
Backticks are generally used when you need to include variables or expressions into a string. This is done by wrapping variables or expressions with ${variable or expression}.
You can also write a quote inside another quote. However, the quote should not match the surrounding quotes.

Access String Characters

You can access the characters in a string in two ways.
One way is to treat strings as an array.
Another way is to use the method [String].charAt([index]).

In JavaScript, strings are immutable. That means the characters of a string cannot be changed.
JavaScript is case-sensitive. That means in JavaScript, the lowercase and uppercase letters are treated as different values.
To use a multiline string, you can either use the + operator or the \ operator.
